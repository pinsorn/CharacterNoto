🎮 Character Manager

A web-based character management system for RPGs, storytelling, and game development

Launch Character Manager View on GitHub

🎭 Character Management

Create and manage characters with avatars, custom parameters, and detailed profiles

🎒 Inventory System

Track items, move them between characters, and manage quantities with ease

🏆 Badge System

Dynamic badges with JavaScript conditions that respond to character stats

📱 Responsive Design

Works perfectly on desktop and mobile with multiple view modes

💾 Data Management

Import/export functionality with automatic localStorage persistence

🌍 Multilingual Support

Built-in support for Thai and other non-English characters

🚀 Quick Start

  1. Launch the app - Click the button above
  2. Add a character - Use "Add Character" or "Add Non-Character"
  3. Customize - Upload avatars, add custom parameters, manage inventory
  4. Create badges - Set up conditional badges based on character stats
  5. Export/Import - Save your data or share with others
